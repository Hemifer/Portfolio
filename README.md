# Personal Portfolio Website

This repository contains the source code for my personal portfolio website. The site serves as a central place to showcase my projects, technical skills, and background as a Computer Science student and developer.

The portfolio was built to provide a clean, interactive way for visitors and recruiters to explore my work and learn more about my experience and interests in software development.

## Overview

The website is designed with a simple and modern layout that focuses on clarity and usability. It includes an animated landing section, a structured overview of my background, and an organized display of development projects.

The main goal of the portfolio is to highlight my technical abilities, demonstrate completed work, and provide an easy way to contact me or view my repositories.

Pages
Home

The landing page introduces who I am and what I do. It includes:

A hero section with a brief introduction

Featured projects

A summary of my background

Quick contact links

### About

The About page provides more detailed information about:

My background and professional mindset

My education in Computer Science

Programming languages and technologies I work with

My interests and motivations as a developer

### Projects

The Projects page showcases several development projects, each linking directly to its GitHub repository. Hover interactions reveal additional descriptions for each project.
